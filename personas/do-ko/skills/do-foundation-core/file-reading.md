# 파일 읽기 최적화

## 크기별 점진적 로딩

- 200줄 미만: 파일 전체 읽기
- 200-500줄: Grep으로 줄 번호 찾은 후 offset/limit으로 Read
- 500-1000줄: 파일 전체 읽기 금지. Grep 먼저, 50-100줄 단위로 읽기
- 1000줄 초과: Read 대신 Grep에 context(-C) 옵션 사용. 전체 이해가 필요하면 Explore 에이전트에 위임

## 실용 규칙

- 읽기 전: Grep으로 관심 있는 정확한 줄 번호 찾기
- 수정 시: 수정할 섹션 + 전후 20줄 컨텍스트만 읽기
- 탐색 시: Grep으로 진입점 찾기부터 시작, 관련 섹션만 읽기

## 토큰 예산

한 줄당 약 10-20 토큰. 1000줄 파일 = 10K-20K 토큰.
10개 파일에서 50줄씩 타겟 읽기 = 5-10K 토큰 (전체 읽기의 50-100K 대비).
